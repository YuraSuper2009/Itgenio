Условие задачи

Один компьютер имеет только три команды: ADD, TIME, MINUS. ADD: прибавляет один к a, TIME: умножает a на два, MINUS: вычитает три из a. Входные данные Входные данные состоят из нескольких наборов входных данных. Первая строка содержит целое число t - количество наборов входных данных. Следующие t строк содержат описание наборов входных данных. От второй до последней строки содержат a и b. a это значение числа до изменения, b после. Выходные данные Выведите t строк, каждая строка должна содержать строку, команду, которая применялась.
